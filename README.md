# Smart Complaint Resolution System

A web-based application that allows users to register complaints and enables admins to manage and resolve them efficiently using priority-based handling.

## Technologies Used
- Java
- Servlet & JSP
- Hibernate
- MySQL
- Apache Tomcat
- Maven
- Git & GitHub

## Features
- User registration and login
- Complaint submission
- Priority-based complaint handling
- Admin complaint management
- Database persistence using Hibernate

## Project Structure
src/main/java  
- com.smartcomplaint.controller  
- com.smartcomplaint.service  
- com.smartcomplaint.dao  
- com.smartcomplaint.model  
- com.smartcomplaint.util  

src/main/webapp  
- JSP pages  

## How to Run
1. Import project into Eclipse as **Maven Project**
2. Configure Tomcat Server
3. Update MySQL credentials in `hibernate.cfg.xml`
4. Run the project on server
5. Access: `http://localhost:8080/SmartComplaintSystem`
